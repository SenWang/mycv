我的 OpenCV 範例
====
Linux下 :<br/>
<ol>
<li>mkdir build</li>
<li>cd build</li>
<li>Linux下: cmake .. </li>
</ol>

Windows下 : <br/>
<ol>
<li>mkdir build</li>
<li>cd build</li>
<li>cmake -D OpenCV_DIR=C:\opencv\build .. </li>
</ol>
